# jqueryplus
### jquery插件,layui类似bootstrap,功能也够用了,其余都是来补充这个的.
- city  一个省市县插件,可以展示省市,配合weui省市县非常棒;
- copy 不需要flash的复制插件
- jq-star 打星评分
- jqzoom 放大镜效果,淘宝看商品
- jscolor 颜色选择框
- validform 最好用的表单验证,使用非常容易
- php.js 用js写成的php函数,非常适合php开发使用,新增js模板函数tpl;
- layer 弹出层,功能超过,包括相册看图
- layui,类似bootstrap,包括了layer所有功能,更加强大
